Vibe Kanban 提供全面的键盘快捷键，帮助您高效地导航和与应用程序交互。这些快捷键按范围和上下文组织。

## 平台键

- **⌘** = Mac Command 键
- **Ctrl** = Windows/Linux Control 键

## 全局快捷键

这些快捷键在应用程序的任何地方都有效（除非您在输入字段中打字）：

| 快捷键 | 操作 | 描述 |
|----------|--------|-------------|
| `C` | 创建任务 | 打开任务表单对话框以创建新任务 |
| `⌘/Ctrl + S` | 聚焦搜索 | 将键盘焦点置于全局搜索输入 |

## 看板导航

当任务卡片用蓝色焦点环高亮显示时，这些快捷键有效：

| 快捷键 | 操作 | 描述 |
|----------|--------|-------------|
| `k` / `j` | 垂直导航 | 在当前列中向上/向下移动焦点 |
| `h` / `l` | 水平导航 | 将焦点移至上一个/下一个非空列中的第一个任务 |
| `Enter` | 打开任务 | 打开聚焦任务的任务详情面板 |
| `d` | 删除任务 | 删除当前聚焦的任务 |

## 表单提交

这些快捷键在各种表单和对话框中有效：

| 快捷键 | 操作 | 上下文 |
|----------|--------|---------|
| `⌘/Ctrl + Enter` | 提交 | 创建并启动（任务表单）、保存（模板编辑器） |
| `⌘/Ctrl + Enter` | 发送消息 | 在任务界面中向编程代理发送消息 |
| `⌘/Ctrl + Enter` | 保存评论 | 在差异审查中添加或保存评论 |
| `Enter` | 新行 | 在消息字段中创建新行 |
| `Shift + Tab` | 切换代理配置文件 | 在代理配置文件之间切换（例如，PLAN 到 DEFAULT） |
| `Escape` | 取消/清除 | 在后续操作中清除草稿，取消评论编辑 |

## 搜索

| 快捷键 | 操作 | 描述 |
|----------|--------|-------------|
| `⌘/Ctrl + S` | 聚焦搜索 | 将光标移至搜索输入字段 |
| `Escape` | 清除搜索 | 清除搜索文本并移除焦点（在搜索字段中时） |

*注意：搜索字段显示"⌘S"徽章以帮助您发现此快捷键。*

## 提示

- **焦点指示器**：查找任务卡片周围的蓝色焦点环，以了解看板导航快捷键何时处于活动状态
- **输入字段**：在文本字段中打字时，全局快捷键被禁用以避免冲突
- **视觉提示**：一些快捷键在界面中显示徽章或提示（如搜索旁边的"⌘S"）
- **修饰键**：除非快捷键专门使用这些修饰符，否则在按住 Ctrl/⌘/Alt 时全局快捷键被禁用
