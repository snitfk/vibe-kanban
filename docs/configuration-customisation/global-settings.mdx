## 主题

在浅色和深色主题之间切换以适应您的偏好。

## 默认代理配置

为新任务尝试选择默认代理和变体。此配置文件在创建新任务尝试和后续操作时预先选择。

1. **选择代理**（例如，Claude Code、Gemini CLI、Codex）
2. **选择变体**（如果可用）（例如，Default、Plan、Router）

<Tip>
您可以在创建尝试对话框中为每次尝试覆盖默认代理配置。
</Tip>

## 编辑器集成

配置与您首选代码编辑器的集成，实现无缝开发工作流程。

### 选择您的编辑器

从各种支持的编辑器中选择：
- **VS Code** - Microsoft 流行的代码编辑器
- **Cursor** - 具有 AI 原生功能的 VSCode 分支
- **Windsurf** - 为协作开发优化的 VSCode 分支
- **Neovim**、**Emacs**、**Sublime Text** - 其他流行的编辑器
- **自定义** - 使用自定义 shell 命令

### 远程 SSH 配置

<Frame>
<img src="/images/vk-editor-ssh.png" alt="Vibe Kanban 设置编辑器部分显示 ssh 配置选项。" />
</Frame>

当在远程服务器上运行 Vibe Kanban 时（例如，通过 Cloudflare 隧道、ngrok 访问或作为 systemctl 服务），您可以配置基于 VSCode 的编辑器通过 SSH 打开项目，而不是假设 localhost。

此功能适用于 **VS Code**、**Cursor** 和 **Windsurf** 编辑器。

#### 何时使用远程 SSH

在以下情况下启用远程 SSH 配置：
- Vibe Kanban 在远程服务器（VPS、云实例等）上运行
- 您通过隧道或反向代理访问 Web UI
- 您的代码文件位于与浏览器不同的计算机上
- 您希望本地编辑器通过 SSH 连接到远程服务器

#### 配置字段

1. **远程 SSH 主机**（可选）
   - 远程服务器的主机名或 IP 地址
   - 示例：`example.com`、`192.168.1.100`、`my-server`
   - 必须可以从本地计算机通过 SSH 访问

2. **远程 SSH 用户**（可选）
   - 用于连接到远程服务器的 SSH 用户名
   - 如果未指定，SSH 将使用您的默认用户或 SSH 配置

#### 工作原理

配置远程 SSH 后，点击"在 VSCode 中打开"（或 Cursor/Windsurf）：
1. 生成特殊协议 URL，如：`vscode://vscode-remote/ssh-remote+user@host/path/to/project`
2. 在默认浏览器中打开，这将启动本地编辑器
3. 您的编辑器通过 SSH 连接到远程服务器
4. 项目或任务工作树在远程上下文中打开

这适用于项目级别和任务工作树打开。

#### 前提条件

- 在本地计算机和远程服务器之间配置 SSH 访问
- 设置 SSH 密钥或凭据（无密码提示）
- 安装 VSCode Remote-SSH 扩展（或 Cursor/Windsurf 的等效扩展）
- 远程服务器路径必须可通过 SSH 访问

<Tip>
首先使用 `ssh user@host` 测试您的 SSH 连接，以确保它在不提示输入密码的情况下工作。
</Tip>

## Git 配置

配置 git 分支命名偏好。

### 分支前缀

为自动生成的分支名称设置前缀（例如，`vk` 导致 `vk/task-name`）。留空表示无前缀。

## 通知

切换音效和推送通知以了解任务状态变化。

## 遥测

启用或禁用遥测数据收集以帮助改进 Vibe Kanban。

## 任务标签

管理全局任务标签以加速所有项目的任务创建。任务标签允许您定义可重用的文本片段，可以使用 @提及插入到任务描述中。

## 代理设置（配置文件和变体）

在**设置 → 代理**下定义和自定义代理变体。变体允许您为同一代理维护多个配置（例如，Claude Code "PLAN" 变体）。

<Card title="代理配置文件和变体" icon="robot" href="/configuration-customisation/agent-configurations">
  配置代理变体的详细指南和示例
</Card>

## 安全和免责声明

管理入门、安全免责声明和遥测通知的确认和重置选项。

- **入门**：重置入门流程以重新运行初始设置。
- **安全免责声明**：重置或查看安全免责声明提示。
- **遥测通知**：重置或查看遥测数据收集确认。
