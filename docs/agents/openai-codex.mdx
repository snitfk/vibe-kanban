title: "OpenAI Codex"
description: "设置 OpenAI Codex 集成"

---

<Steps>
<Step title="运行 OpenAI Codex">
  ```bash
  npx -y @openai/codex
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。按照 [OpenAI 帮助中心](https://help.openai.com/en/articles/11369540-using-codex-with-your-chatgpt-plan)的认证说明，使用您的 ChatGPT 计划使用 Codex。

  或者，可以通过设置 `OPENAI_API_KEY` 环境变量通过 OpenAI API 使用 Codex。有关更多详细信息，请参阅 [OpenAI 文档](https://developers.openai.com/codex/pricing/#use-an-openai-api-key)
</Step>

<Step title="启动 Vibe Kanban">
  认证后，启动 Vibe Kanban：

  ```bash
  npx vibe-kanban
  ```

  您现在可以在创建任务尝试时选择 OpenAI Codex。
</Step>
</Steps>
