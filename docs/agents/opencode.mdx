---
title: "OpenCode"
description: "设置 SST 的 OpenCode"
icon: https://www.vibekanban.com/images/logos/opencode-light.svg
---

<Steps>
<Step title="运行 OpenCode">
  ```bash
  npx -y opencode-ai
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。有关更多详细信息，请参阅 [OpenCode GitHub 页面](https://github.com/sst/opencode)。
</Step>

<Step title="启动 Vibe Kanban">
  认证后，启动 Vibe Kanban：

  ```bash
  npx vibe-kanban
  ```

  您现在可以在创建任务尝试时选择 OpenCode。
</Step>
</Steps>
