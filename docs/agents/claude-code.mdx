---
title: "Claude Code"
description: "设置 Anthropic 的 Claude Code 代理"
icon: https://www.vibekanban.com/images/logos/claude.svg#"
---

<Steps>
<Step title="运行 Claude Code">
  ```bash
  npx -y @anthropic-ai/claude-code
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。有关更多详细信息，请参阅 [Claude Code 文档](https://docs.claude.com/en/docs/claude-code/quickstart)。
</Step>

<Step title="启动 Vibe Kanban">
  认证后，启动 Vibe Kanban：

  ```bash
  npx vibe-kanban
  ```

  您现在可以在创建任务尝试时选择 Claude Code。
</Step>
</Steps>
