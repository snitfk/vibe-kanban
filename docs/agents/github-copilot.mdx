<Steps>
<Step title="运行 GitHub Copilot">
  ```bash
  npx -y @github/copilot
  ```
</Step>

<Step title="按照登录说明操作">
  出现提示时，按照屏幕上的说明使用 `/login` 命令进行认证。有关更多详细信息，请参阅 [GitHub Copilot CLI 文档](https://docs.github.com/en/copilot/how-tos/use-copilot-agents/use-copilot-cli)。
</Step>

<Step title="启动 Vibe Kanban">
  认证后，启动 Vibe Kanban：

  ```bash
  npx vibe-kanban
  ```

  您现在可以在创建任务尝试时选择 GitHub Copilot。
</Step>
</Steps>
