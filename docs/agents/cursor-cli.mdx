<Steps>
<Step title="安装 Cursor Agent CLI">
  ```bash
  curl https://cursor.com/install -fsS | bash
  ```

  使用 `cursor-agent --version` 验证安装。有关更多详细信息，请参阅[官方安装指南](https://docs.cursor.com/en/cli/installation)。
</Step>

<Step title="按照登录说明操作">
  使用 `cursor-agent login` 登录（打开浏览器）。您也可以设置 `CURSOR_API_KEY` 环境变量。完整说明：https://docs.cursor.com/en/cli/reference/authentication
</Step>

<Step title="启动 Vibe Kanban">
  认证后，启动 Vibe Kanban：

  ```bash
  npx vibe-kanban
  ```

  您现在可以在创建任务尝试时选择 Cursor Agent CLI。
</Step>
</Steps>
