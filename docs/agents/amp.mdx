<Steps>
<Step title="运行 Amp">
  ```bash
  npx -y @sourcegraph/amp
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。有关更多详细信息，请参阅 [Amp 用户手册](https://ampcode.com/manual#install)。
</Step>

<Step title="启动 Vibe Kanban">
  认证后，启动 Vibe Kanban：

  ```bash
  npx vibe-kanban
  ```

  您现在可以在创建任务尝试时选择 Amp。
</Step>
</Steps>
