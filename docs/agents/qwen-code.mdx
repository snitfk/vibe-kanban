---

<Steps>
<Step title="运行 Qwen Code">
  ```bash
  npx -y @qwen-code/qwen-code
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。有关更多详细信息，请参阅[官方 Qwen 文档](https://github.com/QwenLM/qwen-code)。
</Step>

<Step title="启动 Vibe Kanban">
  认证后，启动 Vibe Kanban：

  ```bash
  npx vibe-kanban
  ```

  您现在可以在创建任务尝试时选择 Qwen Code。
</Step>
</Steps>
