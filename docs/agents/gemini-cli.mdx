title: "Gemini CLI"
description: "设置 Google Gemini CLI"
icon: https://www.vibekanban.com/images/logos/gemini-logo.svg
---

<Steps>
<Step title="运行 Gemini CLI">
  ```bash
  npx -y @google/gemini-cli
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。有关更多详细信息，请参阅 [geminicli.com](https://geminicli.com/)。
</Step>

<Step title="启动 Vibe Kanban">
  认证后，启动 Vibe Kanban：

  ```bash
  npx vibe-kanban
  ```

  您现在可以在创建任务尝试时选择 Gemini CLI。
</Step>
</Steps>
