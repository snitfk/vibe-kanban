## 安装

<Tabs>
<Tab title="VSCode">
直接从 Visual Studio Code Marketplace 安装：

<Card title="VSCode Marketplace" icon="download" href="https://marketplace.visualstudio.com/items?itemName=bloop.vibe-kanban">
  为 VSCode 安装官方 Vibe Kanban 扩展
</Card>

或者，在 VSCode 中按扩展 ID 搜索扩展：
1. 打开 VSCode
2. 按 `Ctrl+Shift+X`（Windows/Linux）或 `Cmd+Shift+X`（macOS）打开扩展
3. 搜索 `@id:bloop.vibe-kanban`
4. 点击**安装**
</Tab>

<Tab title="Cursor">
对于 Cursor，使用 Open VSX Registry：

<Card title="Open VSX Registry" icon="download" href="https://open-vsx.org/extension/bloop/vibe-kanban">
  从 Open VSX 安装以实现 Cursor 兼容性
</Card>

<Tip>
由于从 Open VSX 的深度链接在 Cursor 中不能可靠工作，最简单的安装方法是在 IDE 中按扩展 ID 搜索。
</Tip>

**安装步骤**：
1. 打开 Cursor
2. 打开扩展面板
3. 搜索 `@id:bloop.vibe-kanban`
4. 安装扩展
</Tab>

<Tab title="Windsurf">
对于 Windsurf，使用 Open VSX Registry：

<Card title="Open VSX Registry" icon="download" href="https://open-vsx.org/extension/bloop/vibe-kanban">
  从 Open VSX 安装以实现 Windsurf 兼容性
</Card>

<Tip>
由于从 Open VSX 的深度链接在 Windsurf 中不能可靠工作，最简单的安装方法是在 IDE 中按扩展 ID 搜索。
</Tip>

**安装步骤**：
1. 打开 Windsurf
2. 打开扩展面板
3. 搜索 `@id:bloop.vibe-kanban`
4. 安装扩展
</Tab>
</Tabs>

## 功能

扩展提供集成的工作区视图，具有三个主要组件：

### 日志视图
- 当前任务的任务尝试列表
- 每个任务尝试中执行的代理步骤

### 差异视图
- 代码更改的并排比较
- 内联评论和审查功能

### 进程视图
- 监控正在运行的任务进程
- 进程状态指示器（运行中、已完成、失败）

### 任务管理
- **任务迭代**：继续迭代当前正在进行的任务
- **状态更新**：实时任务状态同步

## 使用工作流程

<Steps>
<Step title="在 Vibe Kanban 中启动任务">
  导航到项目的看板并创建或选择现有任务。
</Step>

<Step title="打开任务">
  点击任务以打开其详细视图。
</Step>

<Step title="启动 IDE 集成">
  根据您首选的 IDE，点击**"在 VSCode 中打开"**、**"在 Cursor 中打开"**或**"在 Windsurf 中打开"**按钮。

  <Note>
  按钮文本将反映您在 Vibe Kanban 设置中配置的编辑器选择。
  </Note>
</Step>

<Step title="在 IDE 中工作">
  您的 IDE 将在任务的专用工作树中打开，扩展 UI 填充有：
  - 当前任务上下文
  - 实时日志
  - 代码差异
  - 进程监控
</Step>
</Steps>

## 故障排除

### 扩展 UI 为空

**问题**：扩展 UI 显示为空或不显示任务信息。

**解决方案**：确保您在 Vibe Kanban 任务创建的工作树中工作。

<Warning>
扩展只能在 VSCode 在与活动 Vibe Kanban 任务工作树对应的目录中打开时显示任务信息。
</Warning>

**解决步骤**：
1. 验证您通过 Vibe Kanban 任务中的"在 [IDE] 中打开"按钮打开了 VSCode
2. 检查您是否在正确的目录/工作树中
3. 如果问题仍然存在，请重新启动 IDE 并重新尝试工作流程

### 扩展未加载

**问题**：Vibe Kanban 扩展未出现在您的 IDE 中。

**解决方案**：
- 通过在扩展中搜索 `@id:bloop.vibe-kanban` 来验证扩展已安装
- 安装后重新启动 IDE
- 检查您使用的 IDE 版本是否兼容
- 对于 Cursor/Windsurf 用户，确保您从 Open VSX Registry 安装

## 最佳实践

<Tip>
**工作流程优化**：始终从 Vibe Kanban Web 界面启动任务，然后再在 IDE 中打开，以确保正确的上下文和工作树设置。
</Tip>

<Tip>
**性能**：关闭未使用的 IDE 窗口以减少资源使用。
</Tip>
