## 什么是 Vibe Kanban？

Vibe Kanban 是一个 **AI 编程代理编排平台**，帮助开发者规划、审查和安全执行 AI 辅助的编程任务。每个任务在独立的 git 工作树中运行，让您在利用 AI 助手强大功能的同时完全掌控代码库。
