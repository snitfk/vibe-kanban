# Vibe Kanban

> 为开发者设计的可视化项目管理工具,与 git 仓库和编码助手(如 Claude Code 和 Amp)集成。

## 快速开始

无需安装即可立即运行 vibe kanban:

```bash
npx vibe-kanban
```

这将在本地启动应用程序并自动在浏览器中打开。

## 什么是 Vibe Kanban?

Vibe Kanban 是专为开发者设计的现代项目管理工具。它通过看板式任务管理帮助您组织编码项目,同时提供与 git 仓库和 AI 编码助手的强大集成。

### ✨ 主要功能

**🗂️ 项目管理**
- 将 git 仓库添加为项目(现有的或创建新的)
- 自动 git 集成和仓库验证
- 跨所有文件的项目搜索功能
- 每个项目的自定义设置和开发脚本

**📋 任务管理**
- 使用看板式看板创建和管理任务
- 任务状态跟踪(待办、进行中、已完成)
- 丰富的任务描述和注释
- 使用多个 AI 助手执行任务

**🤖 AI 助手集成**
- **Claude**: 高级 AI 编码助手
- **Amp**: 强大的开发助手
- **Echo**: 简单的测试/调试助手
- 创建任务并立即启动助手执行
- 迭代开发的后续任务执行

**⚡ 开发工作流**
- 为每次任务尝试创建隔离的 git worktree
- 查看助手所做更改的差异
- 将成功的更改合并回主分支
- 重新变基任务分支以保持最新
- 手动文件编辑和删除
- 集成的开发服务器支持

**🎛️ 开发者工具**
- 从文件系统浏览和验证 git 仓库
- 在您喜欢的编辑器(VS Code、Cursor、Windsurf、IntelliJ、Zed)中打开任务 worktree
- 实时执行监控和进程控制
- 单独或一次性停止正在运行的进程
- 任务完成的声音通知

## 工作原理

1. **添加项目**: 导入现有 git 仓库或创建新仓库
2. **创建任务**: 定义需要构建或修复的内容
3. **使用 AI 执行**: 让编码助手在隔离的环境中处理您的任务
4. **审查更改**: 使用 git 差异准确查看修改了什么
5. **合并结果**: 将成功的更改合并到您的主代码库中

## 核心功能

Vibe Kanban 通过以下关键功能提供完整的项目管理体验:

**项目仓库管理**
- 用于管理编码项目的完整 CRUD 操作
- 自动 git 仓库检测和验证
- 初始化新仓库或导入现有仓库
- 项目范围的文件搜索功能

**任务生命周期管理**
- 使用丰富的描述创建、更新和删除任务
- 通过可自定义的状态工作流跟踪任务进度
- 一键式任务创建,立即执行 AI 助手
- 具有详细执行历史记录的任务尝试跟踪

**AI 助手执行环境**
- 隔离的 git worktree,用于安全的代码实验
- 实时执行监控和活动日志记录
- 进程管理,能够停止单个或所有进程
- 支持后续执行以迭代解决方案

**代码更改管理**
- 查看任务执行期间所做所有更改的详细差异
- 分支状态监控以跟踪与主分支的偏离
- 一键将成功的更改合并回主分支
- 自动重新变基以保持任务分支最新
- 手动文件删除和清理功能

**开发集成**
- 在您喜欢的代码编辑器中直接打开任务 worktree
- 启动和管理开发服务器以测试更改
- 浏览本地文件系统以添加新项目
- 服务可用性的健康监控

## 配置

Vibe Kanban 通过其配置系统支持自定义:

- **编辑器集成**: 选择您喜欢的代码编辑器
- **声音通知**: 自定义完成声音
- **项目默认值**: 设置默认设置和开发脚本

## 技术架构

- **后端**: Rust 与 Axum Web 框架
- **前端**: React 与 TypeScript
- **数据库**: SQLite 用于本地数据存储
- **Git 集成**: 用于仓库管理的原生 git 操作
- **进程管理**: 基于 Tokio 的异步执行监控

## 要求

- Node.js (用于 npx 执行)
- Git (用于仓库操作)
- 您喜欢的代码编辑器(可选,用于打开任务 worktree)

## 支持的平台

- Linux x64
- Windows x64
- macOS x64 (Intel)
- macOS ARM64 (Apple Silicon)
